# AI Developer Journey

My step-by-step journey learning AI development from scratch.

## Phase 1 — Python Basics ✅
**Project: Data Explorer**
- Loads a CSV dataset of AI models
- Calculates statistics (mean, min, max)
- Identifies best performing model
- Generates a bar chart comparison
- Saves filtered results to new CSV

**Skills learned:** pandas, matplotlib, functions, loops, File I/O

## Phase 2 — Machine Learning ✅
**Project: Movie Rating Predictor**
- Loaded and cleaned 4803 real movies from TMDB dataset
- Trained 3 ML models: Decision Tree, Random Forest, Logistic Regression
- Best model: Logistic Regression (81.0% cross-validated accuracy)
- Key finding: vote_count is the strongest predictor of high ratings
- Generated confusion matrix and feature importance charts

**Skills learned:** scikit-learn, train/test split, cross validation,
classification, model comparison, confusion matrix, feature importance
